# Meeting Caller Mac

macOS menu bar app for automatic meeting control via Meeting Caller ESP32 system.

## Features
- Detects camera usage and auto-starts meetings on ESP32 Master
- Menu bar icon with meeting status, manual start/stop, light control
- Dashboard with live status overview
- Wiz smart plug control (ring light / deco light)
- Configurable: full auto, semi-auto (confirm stop), or manual only
